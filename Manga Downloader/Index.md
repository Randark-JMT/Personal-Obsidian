## 关于"Signal"在Pylance中无法正确解析
这个问题很早之前就被人发现了，并且确定是`Pylance`特有的问题。无论是第三方实现的`PyQt`，还是官方推出的`Pyside`，在大部分版本中都有出现这个问题。
在`Pyside6`中，最为恰当的解决办法就是执行：
```python
pyside6-genpyi all
```
